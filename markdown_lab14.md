## Задание 1: Механизм семафоров

### Описание реализации

Разработан командный файл `semaphore.sh`, реализующий упрощенный механизм семафоров с использованием файловой блокировки.

**Принцип работы:**
- Файл `/tmp/resource.lock` используется как семафор
- Процесс в привилегированном режиме создает блокировку и использует ресурс
- Обычный процесс ожидает освобождения ресурса в течение времени t1
- Время использования ресурса t2 меньше времени ожидания t1

**Параметры:**
- t1 = 10 секунд (время ожидания)
- t2 = 5 секунд (время использования)

### Листинг программы

```bash
#!/bin/bash

LOCK="/tmp/resource.lock"
t1=10
t2=5

if [ "$1" == "privileged" ]; then
    echo "Privileged: taking resource"
    touch $LOCK
    sleep $t2
    rm $LOCK
    echo "Privileged: released"
else
    echo "Normal: waiting for resource"
    waited=0
    while [ $waited -lt $t1 ] && [ -f $LOCK ]; do
        sleep 1
        ((waited++))
    done
    if [ ! -f $LOCK ]; then
        touch $LOCK
        echo "Normal: using resource"
        sleep $t2
        rm $LOCK
    else
        echo "Normal: timeout"
    fi
fi
